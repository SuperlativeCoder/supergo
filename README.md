## MAPPING
```js
|-- base/handlers           // shell/cmd 脚本命令文件
|-- docker                  // Dockerfile
|-- scripts                 // 注册入口
|-- src                 
    |-- base                // 封装
    |-- events              // 事件
    |-- handler             // 执行入口
    |-- help                // 帮助
    |-- interval            // 定时器
    |-- service             // 服务
    |-- listener    // 路由监听
```

## RUN
```shell
    npm run start
```
or
```
    docker build ./docker
```
