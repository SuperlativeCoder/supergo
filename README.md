## Mapping
```js
|-- base/handlers           // shellcmd 脚本命令文件
|-- config                  // 配置
|-- docs                    // 文档
|-- scripts                 // 注册入口
|-- public
|-- src       
    |-- include             // Hubot-Server Kit
    |-- locale              // 国际化
    |-- scripts             // Hubot
        |-- base                // 封装
        |-- handler             // 执行入口
        |-- interval            // 定时器
        |-- listener            // 路由监听
        |-- service             // 服务
    |-- server              // 管理后台
    |-- view                // 管理后台 UI
```
