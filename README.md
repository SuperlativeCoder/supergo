# supergo
a robut project

## MAPPING
```js
|-- base/handlers       // shell/cmd 脚本命令文件
|-- docker              // Dockerfile
|-- scripts             // 入口
|-- src                 
    |-- base            // 封装
    |-- event           // 时间
    |-- handler         // 执行入口
    |-- help            // 帮助
    |-- interval        // 定时器
    |-- service         // 服务
    |-- main.js         // 执行入口
```

## RUN
```shell
    npm run start
```
or
```
    docker build ./docker
```
