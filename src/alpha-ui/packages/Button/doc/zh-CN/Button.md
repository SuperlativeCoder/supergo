## 按钮

> 按钮用于开始一个即时操作
----------

## 使用范例
> 普通按钮
:::demo
```html
<vs-Button>Default</vs-Button>
<vs-Button type="text">Text</vs-Button>
<vs-Button type="ghost">Ghost</vs-Button>
<vs-Button type="primary">Primary</vs-Button>
<vs-Button type="info">Info</vs-Button>
<vs-Button type="success">Success</vs-Button>
<vs-Button type="warning">Warning</vs-Button>
<vs-Button type="error">Error</vs-Button>
```
:::

> 按钮大小
:::demo
```html
<vs-Button type="primary" size="large">Primary (large)</vs-Button>
<vs-Button type="primary" size="default">Primary (default)</vs-Button>
<vs-Button type="primary" size="small">Primary (small)</vs-Button>
<vs-Button type="primary" size="mini">Primary (mini)</vs-Button>
</br>
<vs-Button type="primary" size="mini" long>Primary (mini:long)</vs-Button>
```
:::

> 按钮形状
:::demo
```html
<vs-Button type="primary" shape="round">Primary</vs-Button>
<vs-Button type="info" shape="round">Info</vs-Button>
<vs-Button type="success" shape="round">Success</vs-Button>
<vs-Button type="warning" shape="round">Warning</vs-Button>
<vs-Button type="error" shape="round">Error</vs-Button>
<vs-Button type="primary" icon="search">Primary</vs-Button>
<vs-Button type="primary" icon="search" shape="circle"></vs-Button>
<vs-Button type="primary" icon="android-camera" shape="circle"></vs-Button>
</br></br>
<vs-Button type="primary" shape="round" disabled>Primary</vs-Button>
<vs-Button type="info" shape="round" disabled>Info</vs-Button>
<vs-Button type="success" shape="round" disabled>Success</vs-Button>
<vs-Button type="warning" shape="round" disabled>Warning</vs-Button>
<vs-Button type="error" shape="round" disabled>Error</vs-Button>
<vs-Button type="primary" icon="search" disabled>Primary</vs-Button>
<vs-Button type="primary" icon="search" shape="circle" disabled></vs-Button>
```
:::

> 按钮组
:::demo
```html
<vs-ButtonGroup>
  <vs-Button type="primary">上一页</vs-Button>
  <vs-Button type="primary">下一页</vs-Button>
</vs-ButtonGroup>
<vs-ButtonGroup>
  <vs-Button type="primary" icon="android-arrow-dropleft"></vs-Button>
  <vs-Button type="primary" icon="android-arrow-dropup"></vs-Button>
  <vs-Button type="primary" icon="android-arrow-dropright"></vs-Button>
</vs-ButtonGroup>
```
:::
