## 图片
> 图片相关操作组件，以来七牛平台
-----------

## 使用范例
> 预览
:::demo
```html
<vs-Images
    :width="200" :height="150"
    previewable 
    src="http://pic.sc.chinaz.com/files/pic/pic9/201207/xpic5843.jpg">
</vs-Images>
```
:::

```html
<vs-Images

    scaleable 
    rotaeable 
    previewable 
    src="http://pic.sc.chinaz.com/files/pic/pic9/201207/xpic5843.jpg">
</vs-Images>
```
:::
